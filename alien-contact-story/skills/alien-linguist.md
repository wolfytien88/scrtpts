# Skill: 外星語言學（Alien Linguist）

## 觸發條件
撰寫沃艾人（Voai）或卡洛族（Karo）的溝通場景時啟用。

---

## 沃艾語（Voai-Speak）語法規則

### 核心原則：結果優先
沃艾人的認知模式是「先知道結果，再理解過程」。
這反映在語言結構上：

**人類語序**：「我今天去了市場，買了蘋果。」（原因→行動→結果）
**沃艾語序**：「蘋果已被取得。市場，今日，我。」（結果→地點→時間→行動者）

### 沃艾對話範例

```
艾歐拉說：「合作已被選擇。林，你。理由：生存，雙方。」
（人類翻譯：「你林已選擇合作。理由是雙方的生存。」）

艾歐拉說：「困惑已產生於我。情緒，人類，非效率，但必要。」
（人類翻譯：「我感到困惑。人類的情緒似乎沒有效率，但卻是必要的。」）
```

### 沃艾語的特殊表達
- **無「我想」或「我覺得」**：沃艾人不分離思想與事實
- **情緒用第三人稱描述**：「恐懼已產生於我」而非「我很害怕」
- **時間模糊**：不強調過去/現在/未來，而是「狀態」
- **問句形式**：在陳述句後加「——詢問。」

```
艾歐拉：「地球，修復，可能。——詢問。」
（人類：「地球有可能被修復嗎？」）
```

## 卡洛族（Karo）溝通方式

卡洛族不使用語言溝通，而是以下方式：
- **引力波干擾**：在接近時使設備異常，作為「宣示存在」
- **能量吸收模式**：對目標釋放特定頻率，意思是「你是資源」
- **無談判**：卡洛族不具有「溝通意圖」——它們是真正的他者

*注意：卡洛族在故事中不說話。任何試圖給卡洛族「語言」或「對話」的段落需要審查。*

## 翻譯場景寫法

林曉晨翻譯沃艾語時，讀者看到兩個版本：

```
監測螢幕上的沃艾語音節序列化為文字：
「存活已確認，地球，此刻。威脅已接近，60光年，Karo單位。
 協助已被請求，沃艾，對人類。時間：有限。——詢問：同意？」

林曉晨慢慢翻譯，聲音因震驚而變得平靜：
「它說...地球目前還活著。但有威脅正在靠近，60光年外。
 它在請求我們的協助。然後它問——我們同意嗎？」
```
