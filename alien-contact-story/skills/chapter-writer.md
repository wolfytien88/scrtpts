# Skill: 章節撰寫（Chapter Writer）

## 觸發條件
起草、修改或審查任何章節時啟用。

## 章節結構模板

每章應包含以下結構（不強制按此順序，但需包含所有元素）：

```
[開場鉤子]    ← 前100字，用感官細節或懸念吸引讀者
[場景建立]    ← 建立時間、地點、氛圍
[核心衝突]    ← 本章的主要張力（內在或外在）
[轉折點]      ← 約在章節60–70%處，打破讀者預期
[結尾懸念]    ← 最後一段留下問題或威脅，勾住下一章
```

## 各章特定指引

### 第1章：黃昏地球
- 開場：用林曉晨的視角凝望地球，描寫她的複雜情感
- 核心：建立月球基地的日常，讓讀者熟悉2157年
- 鉤子：最後一幕，監測螢幕出現奇異射線模式

### 第2章：訊號
- 張力來源：科學界的興奮 vs 政界的恐慌
- 使用技術感對話展示林曉晨的專業能力
- 結尾：訊號中出現人類DNA的四個鹼基排列——ACGT

### 第3章：第一道光
- 新聞快報形式插入，展示全球反應
- 重點場景：飛船出現瞬間，林曉晨的感受
- 情緒基調：敬畏 > 恐懼

### 第4章：橋樑
- 突破時刻需要鋪墊：先展示失敗嘗試
- 沃艾語的第一句話：用音節描述，再附翻譯
- 艾歐拉的問題「你們準備好了嗎？」作為章節結尾

### 第5章：使者
- 艾歐拉降落儀式需詳細描述（讓讀者記住）
- 人群反應：展示人類社會的多元（歡迎/恐懼/崇拜）
- 林曉晨與艾歐拉的第一次面對面

### 第6章：沃艾世界
- 用林曉晨的感官一一描述外星環境
- 「奇觀感」是這章的情緒主調
- 在驚嘆中埋下不安：林曉晨注意到奇怪的事

### 第7章：陰影
- 節奏放慢，知識揭露型章節
- 林曉晨讀檔案的內心反應比檔案內容更重要
- 結尾：她決定還是不信任沃艾人

### 第8章：第三方
- 艾歐拉的坦白場景：必須讓讀者相信他的誠意
- 卡洛族的形象：不要讓它們只是「大反派」——給予生態位邏輯
- 林曉晨的選擇：她選擇相信

### 第9章：地球的選擇
- 政治場景需要多聲音：至少展示四種不同立場
- 林曉晨的演講是高潮，但她本人也有懷疑
- 投票結果緊張（非一邊倒）

### 第10章：星際風暴
- 戰鬥場景：重視戰術邏輯，而非單純動作
- 人類和沃艾的配合協調問題是衝突來源
- 陳大維的勇敢表現（為第11章鋪墊）

### 第11章：犧牲
- 陳大維的死：不英雄主義，真實而痛苦
- 林曉晨的情感崩潰與重建
- 人類的韌性讓艾歐拉改變了對人類的看法

### 第12章：新紀元
- 緩慢、深呼吸的節奏
- 展示變化：地球的空氣、人們的眼神
- 最後一頁：新訊號來自更遠的地方——開放結尾

## 節奏控制
- 動作/戰鬥場景：短句、段落短、對話省略詞
- 情感/內省場景：長句、比喻、感官細節豐富
- 技術/政治場景：對話驅動、邏輯清晰、避免冗長說明
